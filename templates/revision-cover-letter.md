# Revision Cover Letter Template

## How to Use This Template

This template is specifically for **resubmission/revision** cover letters, not initial submission cover letters (see `cover-letter.md` for initial submissions). The revision cover letter accompanies your revised manuscript and point-by-point response to reviewers.

1. Replace all placeholders in `[BRACKETS]`
2. Customize the summary of changes to match your actual revisions
3. Reference the separate point-by-point response document
4. Keep the letter concise (ideally 1-2 pages)
5. Have all co-authors review before submission

### Tips for Framing Changes Positively

| Instead of... | Write... |
|--------------|----------|
| "The reviewer was incorrect about..." | "We appreciate the reviewer's perspective and would like to clarify that..." |
| "We disagree with this comment" | "This is an important point. After careful consideration, we have chosen to [approach] because..." |
| "We could not do the requested analysis" | "While the suggested analysis was not feasible due to [reason], we have addressed this concern by [alternative approach]..." |
| "We fixed the errors the reviewer found" | "We thank the reviewer for identifying these issues and have corrected them throughout the manuscript" |
| "We already addressed this in the original manuscript" | "We appreciate this comment. To improve clarity, we have expanded our discussion of this point on Page X, Lines Y-Z" |
| "We deleted the section as requested" | "Following the reviewer's recommendation, we have streamlined the manuscript by [restructuring/condensing] this section" |
| "The reviewer misunderstood our methods" | "We realize our original description may not have been sufficiently clear. We have rewritten the Methods section to better explain..." |

### Key Principles for Positive Framing

1. **Lead with gratitude.** Every paragraph about changes should acknowledge the reviewer's contribution.
2. **Use active, positive language.** "We have strengthened..." not "We were forced to change..."
3. **Show how changes improved the paper.** Frame every revision as making the manuscript better, not just complying with demands.
4. **Be specific about improvements.** "We added a subgroup analysis (new Table 3) that reveals important age-related differences" is better than "We did the analysis."
5. **Avoid defensive language.** Even when you disagree, frame your response as a scientific discussion, not a debate.

### How to Reference the Point-by-Point Response

Always explicitly mention the accompanying response document. Use one of these formulations:

> "A comprehensive point-by-point response to all reviewer comments is provided as a separate document accompanying this cover letter."

> "Please find enclosed our detailed responses to each reviewer comment, including specific references to the revised manuscript pages and line numbers."

> "The accompanying 'Response to Reviewers' document addresses each comment individually, with the revised text quoted for easy reference."

### Mẹo Viết Thư Bằng Tiếng Việt
1. **Luôn giữ thái độ trân trọng** -- Sử dụng các từ ngữ như "Kính gửi", "Trân trọng cảm ơn", "Chúng tôi đánh giá cao".
2. **Ngắn gọn và Cụ thể** -- Tóm tắt các thay đổi chính một cách rõ ràng.
3. **Tham chiếu đến Tài liệu Phản hồi** -- Luôn nhắc đến tài liệu "Response to Reviewers" đính kèm.
4. **Cam kết của tập thể tác giả** -- Khẳng định tất cả các tác giả đều đồng thuận với bản chỉnh sửa.

## Checklist Before Submitting the Revision

### Cover Letter
- [ ] Editor name and journal name are correct
- [ ] Manuscript ID matches
- [ ] All major changes are summarized
- [ ] Point-by-point response document is referenced
- [ ] All authors are listed and have approved
- [ ] Tone is positive and grateful throughout
- [ ] Letter is concise (1-2 pages maximum)

### Accompanying Documents
- [ ] Revised manuscript with track changes
- [ ] Clean version of revised manuscript (if required)
- [ ] Point-by-point response to reviewers (see `response-to-reviewers.md`)
- [ ] Updated supplementary materials (if applicable)
- [ ] Updated figures/tables (if applicable)
- [ ] Completed revision checklist (if journal provides one)

### Final Verification
- [ ] All co-authors have reviewed the revision package
- [ ] Page/line numbers in the response match the revised manuscript
- [ ] No tracked changes remain in the "clean" version
- [ ] All new references are properly formatted
- [ ] Word count is within journal limits
- [ ] Figures meet journal resolution/format requirements
