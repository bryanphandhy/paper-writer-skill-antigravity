# Clinical Trial Registration Guide

> ICMJE requires prospective registration of all clinical trials before the first patient is enrolled. Failure to register may result in desk rejection.

---

## Definition

**Clinical trial** (ICMJE definition): Any research project that prospectively assigns people to an intervention to evaluate the effect on a health outcome.

This includes:
- Drug trials (all phases)
- Device trials
- Surgical procedure comparisons
- Behavioral interventions
- Dietary interventions
- Rehabilitation protocols

This does NOT include:
- Purely observational studies
- Retrospective analyses
- Case reports
- Systematic reviews

## Japan-Specific Registries


- **URL**: https://www.umin.ac.jp/ctr/
- **Language**: Vietnamese and English
- **Cost**: Free
- **Required for**: Many Vietnamese journals, ethical review boards
- **Registration ID format**: UMIN000XXXXXX


- **URL**: https://jrct.niph.go.jp
- **Language**: Vietnamese (with English fields)
- **Required for**: Studies under Japan's Clinical Research Act（臨床研究法）
- **Registration ID format**: jRCTxxxxxxxxxx
- **Mandatory for**: Specified clinical research（特定臨床研究）

### Which to use?

| Study Type | Registry |
|-----------|----------|
| 特定臨床研究（Clinical Research Act） | jRCT（義務） |
| 医師主導治験 | jRCT + ClinicalTrials.gov recommended |
| 国際共同研究 | ClinicalTrials.gov + UMIN-CTR |

## Registration Timing

### ICMJE Requirement
- Register BEFORE first participant is enrolled
- Registration after enrollment = "retrospective registration"
- Many journals will still consider retrospectively registered trials but may require explanation

### What to Register

WHO Trial Registration Data Set (minimum 24 items):
1. Primary registry and trial ID
2. Registration date
3. Secondary IDs
4. Funding source
5. Primary sponsor
6. Secondary sponsor(s)
7. Contact for public queries
8. Contact for scientific queries
9. Public title
10. Scientific title
11. Countries of recruitment
12. Health condition studied
13. Intervention(s)
14. Key inclusion/exclusion criteria
15. Study type
16. Date of first enrollment
17. Target sample size
18. Recruitment status
19. Primary outcome(s)
20. Key secondary outcome(s)
21. Ethics review
22. Completion date
23. Summary results
24. IPD sharing statement

## Reporting Registration in Papers

### In the Abstract
```
Trial Registration: ClinicalTrials.gov Identifier: NCT01234567
```

### In the Methods
```
This trial was registered prospectively at ClinicalTrials.gov
(NCT01234567) on [date], before the first participant was enrolled
on [date].
```

### On the Title Page
Include the registration number. Some journals require it as a separate field.

## Common Issues

| Issue | Solution |
|-------|---------|
| Forgot to register before enrollment | Register immediately; explain in cover letter; some journals accept with justification |
| Outcomes differ from registration | Document and justify all changes; report both registered and actual outcomes |
| Multiple registrations | Cross-reference all IDs; designate one as primary |
| Results posting required | ClinicalTrials.gov requires results within 12 months of completion (US law) |
| Registration expired/outdated | Update registration before submission |
