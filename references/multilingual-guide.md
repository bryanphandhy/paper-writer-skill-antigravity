# Multilingual Paper Writing Guide (English + Vietnamese)

## Overview

This guide supports researchers writing papers in both English and Vietnamese, whether for international journals (English primary) or Vietnamese domestic journals (Vietnamese primary). It covers parallel writing workflows, translation strategies, and formatting conventions for both languages.

---

## Parallel Writing Workflow

### Strategy 1: English First, Then Vietnamese (Recommended for International Papers)

Best when targeting English-language journals with a Vietnamese abstract required.

```
Phase 1: Draft all sections in English
Phase 2: Internal review in English
Phase 3: Translate abstract and key sections to Vietnamese
Phase 4: Native check for Vietnamese sections
Phase 5: Final alignment check (both versions say the same thing)
```

### Strategy 2: Vietnamese First, Then English (For Vietnamese Journals or When Thinking in Vietnamese)

Best when the primary audience is Vietnamese, or when the author thinks more clearly in Vietnamese.

```
Phase 1: Draft all sections in Vietnamese
Phase 2: Internal review in Vietnamese
Phase 3: Translate to English (or have translated)
Phase 4: Native English check / professional editing
Phase 5: Final alignment check
```

### Strategy 3: Simultaneous Bilingual Drafting

Best for experienced bilingual writers. Write each section in your stronger language for that content type.

```
- Methods: Often easier in English (standard terminology)
- Discussion: May be easier in Vietnamese first (complex reasoning)
- Abstract: Write in primary language first, then translate
```

## Vietnamese Academic Writing Conventions

### Register: Formal Academic Tone

Vietnamese academic papers use a formal, objective tone. Personal pronouns like "Tôi" or "Chúng tôi" should be minimized unless absolutely necessary to claim specific agency (e.g., in Methodology).

| Context | Recommended | Avoid |
|---------|-------------|-------|
| Statement | Kết quả cho thấy... / Nghiên cứu chỉ ra rằng... | Chúng tôi nghĩ rằng... / Theo ý kiến của tôi... |
| Negative | Không có sự khác biệt có ý nghĩa thống kê... | Không khác nhau lắm... |
| Purpose | Mục tiêu của nghiên cứu này là... | Chúng tôi muốn làm nghiên cứu này để... |
| Future | Cần thêm các nghiên cứu... | Chúng ta cần làm thêm nghiên cứu... |

### Common Academic Expressions

| English | Vietnamese Academic Expression |
|---------|------------------------------|
| We investigated... | Chúng tôi tiến hành khảo sát... / Nghiên cứu được tiến hành để... |
| It is suggested that... | Kết quả này gợi ý rằng... / Điều này cho thấy... |
| It has been reported that... | Theo các báo cáo trước đây... / Các nghiên cứu trước cho thấy... |
| The purpose of this study was... | Mục tiêu của nghiên cứu này là... |
| In conclusion... | Tóm lại... / Kết luận, ... |
| There was a significant difference | Có sự khác biệt có ý nghĩa thống kê (p < 0.05) |
| No significant difference was observed | Không ghi nhận sự khác biệt có ý nghĩa thống kê |
| Further studies are needed | Cần có thêm các nghiên cứu trong tương lai... |
| To the best of our knowledge | Theo hiểu biết của chúng tôi... |
| The limitations of this study include | Một số hạn chế của nghiên cứu này bao gồm... |

### Thuật ngữ tiếng Anh đi kèm (English Annotation of Technical Terms)

When writing in Vietnamese, include the English term in parentheses at first use if the translation is non-standard or easily confused:

```
Hệ thống hỗ trợ chẩn đoán dựa trên trí tuệ nhân tạo (artificial intelligence - AI) đã được đánh giá.
Mô hình học sâu (deep learning) được sử dụng là mạng nơ-ron tích chập (convolutional neural network - CNN).
```

## Common Translation Pitfalls

### Translation from English to Vietnamese Common Errors

| English Pattern | Wrong Vietnamese (Literal/AI) | Correct Vietnamese |
|----------------|----------------|------------------|
| Significantly | Một cách đáng kể | Có ý nghĩa thống kê (trong ngữ cảnh P-value) |
| We found that... | Chúng tôi đã tìm thấy rằng... | Chúng tôi nhận thấy... / Kết quả cho thấy... |
| Robust results | Kết quả mạnh mẽ | Kết quả đáng tin cậy / Kết quả nhất quán |
| Address the issue | Giải quyết vấn đề (nếu chưa giải quyết xong) | Đề cập đến vấn đề / Đánh giá vấn đề |
| State-of-the-art | Nghệ thuật tiên tiến nhất | Tiên tiến nhất / Hiện đại nhất |

### Passive Voice Conventions

- English medical papers: Passive voice is very common ("Blood samples were collected").
- Vietnamese medical papers: Active voice with a general subject is often more natural ("Tiến hành lấy mẫu máu" rather than "Mẫu máu đã được lấy bởi chúng tôi"). However, passive voice without an explicit agent is acceptable ("Mẫu máu được thu thập...").

## Section-by-Section Translation Checklist

### Title
- [ ] English title: concise, informative, no abbreviations
- [ ] Vietnamese title: matches English meaning exactly, uses appropriate Sino-Vietnamese terminology (Từ Hán Việt)
- [ ] Both titles follow target journal format

### Abstract
- [ ] Structure matches (Background/Methods/Results/Conclusions or Đặt vấn đề/Phương pháp/Kết quả/Kết luận)
- [ ] Word count within limits for both languages
- [ ] Numbers and statistics identical in both versions
- [ ] Key terms consistent with terminology table

### Introduction
- [ ] Same references cited in both versions
- [ ] Background context equivalent (not just translated word-for-word)

### Methods
- [ ] Statistical methods described identically
- [ ] Ethics approval statement present in both

### Results
- [ ] All numerical values identical
- [ ] Table/figure references match
- [ ] Statistical values (p-values, CIs) identical

### Discussion
- [ ] Key arguments present in both versions
- [ ] Limitations section equivalent
- [ ] Conclusion statements aligned

### References
- [ ] Same references in both versions
- [ ] Format follows respective journal guidelines

## Vietnamese Journal Formatting Differences

### Common Differences from International Journals

| Aspect | International (English) | Vietnamese Journals |
|--------|------------------------|-------------------|
| Author names | First Last | Họ và Tên (Full name) |
| Affiliations | Department, Institution, City, Country | Khoa, Bệnh viện/Trường, Thành phố |
| Abstract | English only (usually) | Tiếng Việt + English (both required) |
| Keywords | English | Từ khóa (Tiếng Việt) + Keywords (English) |
| Figures/Tables | English captions | Vietnamese captions (sometimes bilingual) |
| References | Vancouver or APA style | Vancouver (often modified by Ministry of Health/Bo Y Te guidelines) |
| Ethics statement | IRB approval number | Số quyết định Hội đồng Đạo đức |

### Bilingual Abstract Format (Common in Vietnamese Journals)

```
【TÓM TẮT】
Mục tiêu: [Objective]
Đối tượng và Phương pháp: [Methods]
Kết quả: [Results]
Kết luận: [Conclusions]
Từ khóa: [keyword1], [keyword2], [keyword3]

【ABSTRACT】
Objective: [Objective]
Methods: [Methods]
Results: [Results]
Conclusions: [Conclusions]
Keywords: [keyword1], [keyword2], [keyword3]
```
